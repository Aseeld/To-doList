# To-doList

 _To-do List, a small website with a list of tasks you need to complete or things that you want to do._


# Built with:
------------------
- Django
- HTML
- CSS 
- Sqlite3

# Features:
---------------------------------
- Add new list type.
- Add notes to each list.
- Edit notes.
- Delete list.
- Delete note.
